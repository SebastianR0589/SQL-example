# SQL Queries Sample

This project is a collection of SQL queries created to practice and demonstrate core relational database concepts, including table design, data manipulation and CRUD operations.

The focus of this repository is on writing clear and structured SQL queries for common use cases rather than building a full application.

---

## Project Goals
- Practice relational database modeling  
- Create and manage SQL tables  
- Perform CRUD operations using SQL queries  
- Work with realistic sample data  

---

## Contents
- Table creation scripts  
- Sample data inserts  
- Query examples for selecting, updating and deleting data
